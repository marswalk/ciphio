A brief history and what's to come.

Ciphers hide truth. They twist words into noise. For centuries, kings and generals relied on them. But every lock has a key. And every code can break.

We built tools to crack them. No magic. Just logic, math, and code.

The National Cipher Challenge

In 2021, we joined the British National Cipher Challenge. Real ciphertexts. Hard deadlines.

Before LLMs were everywhere, we got stuck. Simple substitution ciphers beat us. So, we evolved. We wrote code to fight code.

Cipher Basics

A cipher is an algorithm. It performs encryption or decryption. It follows well-defined steps.

Plaintext + Key = Ciphertext.

Think of the Caesar shift. A becomes D. B becomes E. Simple mapping. Easy to do. Easy to break.

The Manual Approach

Languages have patterns. The English language is predictable. E is common. Z is rare.

We use Frequency Analysis. Count the letters. Match the peaks. Guess the key.

It works. But it is slow. It requires human intuition. We wanted speed.

Automation

We built automated solvers. First idea: Brute force. Try every key.

Problem? A basic substitution cipher has 26 factorial keys. That is 403 septillion combinations. Checking 1,000 keys a second takes 63 million centuries.

Brute force fails. We need a smarter score. Enter Text Scoring.

We score text using n-grams (bigrams, quadgrams). If a decrypted string has "TH", it scores high. If it has "XZ", it scores low.

Optimization

To find the best key fast, we use Hill Climbing.

Pick a random key. Score it. Swap two letters. Score again. Keep the better one. Climb until you hit the peak.

Sometimes you get stuck in a "local maximum". The score is good, but the text is gibberish. The fix? Simulated Annealing.

We introduce chaos. Sometimes, we accept a worse key to escape the trap. As time passes, we lower the "temperature" (chaos). It settles on the true global maximum. The real plaintext.


def hill_climb(ciphertext):
    best_key = random_key()
    best_score = score(decrypt(ciphertext, best_key))
    
    while True:
        new_key = swap_two_letters(best_key)
        new_score = score(decrypt(ciphertext, new_key))
        
        if new_score > best_score:
            best_score = new_score
            best_key = new_key
